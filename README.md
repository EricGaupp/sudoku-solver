# Sudoku Solver

A basic Sudoku Solver that uses a recursive backtracking algorithm to solve puzzles.

Demo can be found at https://ericgaupp.github.io/sudoku-solver

### Technologies

- TypeScript
- React
- MobX
- Material-UI
